# HvH Agressive Conference

## Запуск
```
pip install -r requirements.txt
python app.py
```
Открой http://localhost:5000

## Админ панель
URL: `/admin`
Пароль: `123wue123`

## Возможности
- Регистрация/вход пользователей
- Личный кабинет с балансом, рейтингом, местом
- Активация ключей для пополнения баланса (ключи создаёт админ)
- Покупка конфигов за баланс
- Пользователи могут выставлять свои конфиги на продажу
- Отзывы
- Рейтинг игроков (управляется в админке)
- Тексты главной и "О HvH" редактируются в админке
- Загрузка аватарок для игроков через админку
